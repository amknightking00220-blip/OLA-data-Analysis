# OLA-data-Analysis
This project analyzes OLA ride bookings data using SQL and POWER BI.
The aim was to acheive solutions for buisness related questions like :

Which vehicle type got the maximum ratings

How many rides cancelled due to specific reason 

Who booked highest number of rides

I collected insights about ride distribution,cancelletion trend, vehicle perfomance.

-- Tools used :
SQL       -   analysing data
POWER BI  -   visualising data
EXCEL     -   importing and cleaning data 

-- The dataset includes ride-level information such as:

Booking ID

Customer ID

Vehicle Type

Ride Distance

Booking Status (Successful / Cancelled)

Cancellation Reason

Driver Ratings

Customer Ratings

-- POWER BI DASHBOARD includes:
Count of Customer ID by Day and Booking Status
Vehicle type and ride distance
Vehicle type and customer rating by day
Customer's and driver's reason for cancellation
